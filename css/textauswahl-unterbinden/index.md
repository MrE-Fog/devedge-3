## Textauswahl unterbinden

Das folgende Beispiel zeigt, wie man die Textauswahl auf einer Webseite mit Hilfe von CSS unterbinden kann.

```css
body {
  user-select: none;
  -moz-user-select: none;
  -khtml-user-select: none;
  -webkit-user-select: none;
}
```
