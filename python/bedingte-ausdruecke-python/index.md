## Bedingte Ausdrücke (englisch: Conditional expressions) (Python)

```python
#!/usr/bin/env python
# -*- coding: utf8 -*-

print "Content-type: text/html\n\n"

x = 1

print ("x ist gleich 1" if x == 1 else "x ist unglich 1")
```
