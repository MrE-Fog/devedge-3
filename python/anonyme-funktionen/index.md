## Anonyme Funktionen

### Beispiel #1

```python
#!/usr/bin/env python
# -*- coding: utf8 -*-

print "Content-type: text/plain\n\n"

f = lambda x: x + 10

print f(2) # Ausgabe: 12
```

### Beispiel #2

```python
#!/usr/bin/env python
# -*- coding: utf8 -*-

print "Content-type: text/plain\n\n"

f = lambda x, y: x + y

print f(2, 10) # Ausgabe: 12
```
