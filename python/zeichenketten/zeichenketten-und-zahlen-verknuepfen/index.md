## Zeichenketten und Zahlen verknüpfen

### Beispiel #1

```python
#!/usr/bin/env python
# -*- coding: utf8 -*-

print "Content-type: text/plain\n\n"

print "Das ist ein Text. Das ist eine Zahl: " + str(5) # Ausgabe: Das ist ein Text. Das ist eine Zahl: 5
```
